# AI-Based Exercise Form Analysis Platform

Full-stack MERN + Python FastAPI project that analyzes workout videos, computes form scores using a contrastive learning model, and returns visual + statistical feedback.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express, MongoDB
- AI Service: FastAPI, PyTorch, MediaPipe Pose

## Features
- Video upload and streaming
- Pose estimation and AI-based form scoring
- Joint-level feedback and analytics (graphs, CSV)
- Modern responsive UI
